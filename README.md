# AI Text Detector
Machine Learning web application that detects whether a given text is AI-generated or human-written.

## 📌 Project Description
AI Text Detector is a machine learning web application that analyzes a given text and predicts whether the text is AI-generated or human-written.

## 🚀 Features
- Detects AI-generated text
- Detects human-written text
- Web interface for easy text input
- Confidence score for prediction

## 🛠 Technologies Used
- Python
- Scikit-learn
- TF-IDF Vectorizer
- Flask
- HTML
- CSS
- JavaScript

## 📊 Model
The model is trained using a text dataset and uses TF-IDF vectorization with a machine learning classifier to predict the probability of AI-generated text.

## ▶ How to Run the Project

1. Clone the repository
   git clone https://github.com/karunyaa-5/AI-Text-Detector.git
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py
4. Then open the browser and go to:
    http://127.0.0.1:5000
